## Опис програмного веб-застосунку психологічного щоденника
Цей веб-застосунок функціонує як щоденник для контролю власного емоційного стану та щоденних записів, а також для створення цілей та досягнень.

## Стек технологій
- **Мова програмування:** C# .NET
- **Фреймворк:** ASP.NET MVC
- **Система управління базами даних:** MS SQL Server
- **Шаблони сторінок:** Razor Pages
- **Фронтенд:** JavaScript, HTML, CSS
- **ORM (Object-Relational Mapping):** Entity Framework

## Основні функції та можливості

1. **Створення облікового запису:**
   Користувачі можуть створити власний обліковий запис з унікальним ім'ям та паролем для забезпечення конфіденційності своїх записів.

2. **Щоденні записи:**
   Користувачі можуть вести щоденний журнал своїх почуттів, емоцій та подій. Це може бути текстовими записами, фотографіями, голосовими записами або навіть емоційними виразами, такими як емодзі.

3. **Аналіз емоцій:**
   Додаток може використовувати аналітику емоцій для відображення змін у власному емоційному стані користувача.

4. **Встановлення цілей і досягнень:**
   Користувачі можуть створювати особисті цілі та завдання для самовдосконалення.

5. **Графіки та статистика:**
   Психологічний щоденник може надавати користувачам візуальну статистику їхніх емоційних змін та досягнень за допомогою графіків і діаграм.

6. **Нагадування:**
   Додаток може надсилати користувачам нагадування про щоденний журнал або просяти оновити свої цілі і завдання.

7. **Приватність і безпека:**
   Застосунок повинен забезпечувати високий рівень приватності та безпеки даних, включаючи шифрування і можливість видалення даних на вимогу користувача.

8. **Спільнота та підтримка:**
   Додаток може надавати можливість користувачам об'єднуватися у спільноти для обміну досвідом і підтримки.

9. **Мобільна версія:**
   Додаток також може мати мобільну версію, щоб користувачі мали доступ до свого психологічного щоденника в будь-який час і в будь-якому місці.

# Опис програмного застосунку "Business Process Modeling Tool(AnalogBpWinDFD)"

## Огляд

Проект "Business Process Modeling Tool" є програмою для моделювання бізнес-процесів. Вона написана на мові програмування C# з використанням фреймворка WPF (Windows Presentation Foundation). Головною функціональністю цього застосунку є можливість створення та аналізу DFD-діаграм (діаграм потоків даних) для візуалізації та оптимізації потоків даних у бізнес-процесах.

## Функціональність

Основні можливості програмного застосунку включають:

- **Створення DFD-діаграм:** Користувач може створювати нові DFD-діаграми для моделювання різних бізнес-процесів.

- **Додавання елементів:** В програмі реалізована можливість додавати елементи, такі як процеси, потоки даних, сутності та джерела даних, на діаграму.

- **З'єднання елементів:** Користувач може встановлювати зв'язки між елементами для визначення потоків даних.

- **Аналіз даних:** Програмний застосунок дозволяє аналізувати потоки даних та виявляти можливі проблеми чи покращення в бізнес-процесах.

- **Експорт імпорт:** Користувач може зберігати та завантажувати свої проекти для подальшої роботи, а також експортувати DFD-діаграми у різних форматах.

## Стек технологій

Програмний застосунок "Business Process Modeling Tool" побудований на основі наступних технологій:

- **Мова програмування:** C#

- **Графічний інтерфейс:** Windows Presentation Foundation (WPF)

- **Збереження даних:** Вбудована система збереження даних та можливість експорту/імпорту даних.

- **Інтеграція з мовами та інструментами моделювання:** Додаткова інтеграція з іншими інструментами моделювання бізнес-процесів (за необхідності).

## Вимоги до середовища

Для запуску програмного застосунку потрібно наявність операційної системи Windows та встановленого .NET Framework.

# Додаток "Веб-сайт OurSite"

Цей веб-сайт розроблений для особистого використання і призначений для збереження спогадів. Він надає можливість зберігати і відображати особисті нотатки, фотографії та інші приємні спогади для вашого відносини.

## Стек технологій

- **Фронтенд:**
  - HTML
  - CSS
  - JavaScript

- **Бекенд:**
  - PHP

- **Система управління базами даних:**
  - MySQL

- **Інструмент для адміністрування бази даних:**

## Основні функції

1. **Додавання і редагування нотаток**: Користувач може створювати та редагувати особисті нотатки та записи про своє відношення.

2. **Завантаження фотографій**: Користувач може завантажувати фотографії і зберігати їх на сайті, а також прикріплювати до нотаток.

3. **База даних**: Всі дані, такі як нотатки та фотографії, зберігаються в базі даних MySQL, що дозволяє ефективно зберігати та керувати ними.

4. **Авторизація та аутентифікація**: Щоб забезпечити приватність користувача, доступ до сайту вимагає авторизації.

5. **Інтерфейс з веб-сервером**: Веб-сайт використовує PHP для взаємодії з сервером та базою даних.

## Опис
Цей програмний застосунок - це антивірусна гра, в якій гравець керує антивірусним програмним забезпеченням, що бореться з вірусами у системі. Гра є платформером, написана на мові програмування C# з використанням платформи Unity.

## Можливості
- Геймплей: Гра пропонує гравцеві інтерактивний досвід, де він керує антивірусом у вигляді персонажа і здійснює пошук та видалення вірусів у віртуальній системі.

- Різноманітність вірусів: Гравцю доведеться боротися з різними видами вірусів, кожен з яких може мати свої унікальні властивості та методи атаки.

- Удосконалення та апгрейди: Гравцеві надаються можливості покращувати свого антивіруса, додавати нові функції та інструменти для боротьби зі зростаючими загрозами.

- Рівні складності: Гра може мати різні рівні складності, від початкового до високого рівня, що дає гравцям різний виклик.

- Графіка та анімація: За допомогою Unity гра може мати відмінну графіку та анімації, що допомагає створити захопливу візуальну атмосферу.

## Стек технологій
- Мова програмування C#: Всі ігрові механіки, логіка та інтеракція з гравцем програмуються на мові C#.

- Unity: Unity є відмінною платформою для створення ігор, яка надає інструменти для розробки 2D та 3D ігор. Вона забезпечує гнучкість та можливість розгортання гри на різних платформах.

- Графічний дизайн і анімація: Для створення візуального ефекту гри використовуються інструменти для розробки графічних ресурсів та анімації, такі як Adobe Photoshop, Blender, а також можливості Unity для анімації об'єктів.

- Звук та музика: Для створення атмосфери в грі додаються звукові ефекти та музика, які можуть бути створені або імпортовані зовнішніми засобами, такими як Audacity або DAW (Digital Audio Workstation).

- Можливості для розробки мультиплатформених ігор: Unity надає можливість розгортати гру на різних платформах, таких як Windows, macOS, Android, iOS тощо.
